# Natural Gas Price Analysis
### JPMorgan Chase - Quantitative Research Virtual Experience

## Project Overview
This project focuses on analyzing historical natural gas 
prices and building a price estimator that can predict 
the cost of natural gas on any given date. The model uses 
seasonal trends and curve fitting techniques to support 
the pricing of commodity storage contracts.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- SciPy

## What I Did
- Explored and visualized historical natural gas price 
  data from 2020 to 2024
- Identified seasonal patterns showing higher prices 
  in winter and lower prices in summer
- Built a seasonal model to fit the historical data
- Created a function that returns a price estimate 
  for any input date
- Extrapolated prices one year into the future
